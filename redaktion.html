Code für programm-daten.js

So setzt du es ein (einmal eingerichtet, dann immer gleich):
  1. „Code erzeugen" klicken, dann „Kopieren".
  2. Datei assets/programm-daten.js öffnen.
  3. Den Block von programm: [ bis ], markieren und durch das Kopierte ersetzen.
  4. Speichern, hochladen — Web & Ausdruck sind sofort aktuell.
